Instruction pour connexion API Spotify

  1. Créez une application sur Spotify Developer Dashboard
  2. Récupérez votre Client ID et Client Secret
  3. Remplacez VOTRE_CLIENT_ID et VOTRE_CLIENT_SECRET dans le fichier JavaScript
  4. Lancez un serveur local et testez la recherche !

Concepts techniques abordés

  • OAuth 2.0 - Authentification Client Credentials
  • API REST - Requêtes GET avec en-têtes
  • JSON - Manipulation de données complexes
  • Pagination - Gestion des résultats multiples
